# nerfstudio Install

[Official guide](https://docs.nerf.studio/quickstart/installation.html)

Make sure to use VS2022 17.8, as newer versions will throw errors when installing tiny-cuda-nn

Download 17.8 via this [link](https://aka.ms/vs/17/release.ltsc.17.8/vs_enterprise.exe) ([direct link](https://download.visualstudio.microsoft.com/download/pr/bd2300cc-5700-4b59-955d-4e5ef66b964a/3fb9aec96707d61e06a38a39d9ba30c6653394747f73d45dfe5240f77e0b2600/vs_Enterprise.exe))

Then, the command to use:
```
vs_Enterprise.exe install --channelUri https://aka.ms/vs/17/release.LTSC.17.8/channel --productId Microsoft.VisualStudio.Product.Community
```
